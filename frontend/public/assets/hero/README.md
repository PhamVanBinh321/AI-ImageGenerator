# Hero Images

Đặt 3 hình ảnh sau vào thư mục này:

1. `cat-astronaut.jpg` - Mèo trong bộ đồ phi hành gia, nền không gian tối
2. `cat-bar.jpg` - Mèo mặc yếm xanh, uống bia trong quán bar ấm áp
3. `farmer-buffalo.jpg` - Nông dân cày ruộng với trâu, nền cảnh đồng xanh

**Lưu ý:**
- Nén ảnh dưới 200KB để tối ưu tải trang
- Đảm bảo ảnh có tỷ lệ phù hợp (khuyến nghị: 16:9 hoặc 21:9)
- Sử dụng format JPG hoặc WebP để giảm kích thước file
